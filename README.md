# Battleship - TDD Implementation

Este proyecto es una implementación del clásico juego "Batalla Naval" desarrollado en **JavaScript moderno (ES6+)**, siguiendo la metodología **TDD (Test-Driven Development)**

El enfoque principal de este repositorio es demostrar la creación de lógica de negocio robusta mediante pruebas automatizadas antes de la implementación de la interfaz de usuario.

## Características Técnicas

- **_Metodología TDD:_** Desarrollo guiado por pruebas usando **Jest** para asegurar un código libre de errores desde la base.
- **_Programación Orientada a Objetos (POO):_** Arquitectura basada en clases desacopladas (_Ship, Gameboard, Player_).
- **_Infraestructura Profesional:_** Configuración de entorno con **Webpack** y gestión de dependencias con `npm`.
- **_Algoritmos de Juego:_** Lógica para el posicionamiento aleatorio de barcos y sistema de turnos.

## Tecnologías Utilizadas

- **_Lenguaje:_** JavaScript (ES6+)
- **_Testing Framework:_** [Jest](https://jestjs.io/)
- **_Bundler:_** Webpack
- **_Estilos:_** CSS3 (Flexbox, Grid)
